Mimir's Well

FV-1 DSP Pedal Platform

Mimir's Well is a guitar pedal powered by the extremely popular FV-1 DSP (digital signal processor). It allows you to run audio processing programs that create amazing reverb, delay, pitch, and plenty of other effects. Some programs provide a single effect with controls to dial in their details, while other programs layer multiple effect types to create unique and interesting soundscapes.

This is the same chip that powers pedals such as the EQD Rainbow Machine and Dispatch Master, EHX Holy Stain, Walrus Arp 87 and Fathom, Black Hole Symmetry, and tons more.

While those other pedals are standalone, each running only one program (or using a selector knob to toggle between a few similar programs), Mimir's Well lets you load up to 16 custom programs you can switch between, in addition to the 7 great, built-in effects.
